# List（列表）、Dict（字典）的使用技巧
## list
### list基本操作
列表是数据容器的一类，是可以一次性储存多个数据的
#### 定义：
列表名称=[元素一，二，三]    eg:s=[43,23,45]
#### 特点：
可以储存不同类型的元素，元素可以重复，有序，可以修改（直接在下面进行处理就行）
#### 获取元素：
正向索引：s[0]，反向索引：s[-1]（倒数第一个）
正向索引从0开始，反向索引从-1开始
#### 删除指定元素：
del s[n]
#### 遍历元素：
```
for item in s:
    print(item)
```
### 切片
指的是对于操作数据截取其中一部分的操作，列表，字符串，元组都支持切片
#### 语法：
序列数据[开始索引：结束索引：步长]
不包含结束索引位置对应的元素（开始索引未指定默认为0，结束索引默认为列表长度，包前不包后）
正向反向索引都可以，eg:print(s[0:-2:1])
步长是选取间隔，步长默认为1。步长为n，跳过n-1个元素
从大列表中截取小列表，所以还是类型为list
注意冒号不能省略

### 方法
列表的常用方法就是指列表这类数据类型内置的常见功能（添加元素，删除元素，排序等）
#### 常见方法：
append()   在尾部追加元素
insert()   在指定索引之前插入该元素      eg:s.insert(0,92) 0号元素之前插入元素92
remove()    移除值
pop()        删除指定索引的元素（默认的是最后一个）
sort()        对列表进行排序（要求：类型一样）
reverse()        反转列表元素
#### 题目分析：
##### 题目一
```
s=[]   //空列表
for i in range(10):     // 遍历函数
    num = int(input("请输入一个有效的数据："))   //直接把输入的值变成整数
    s.append(num)      //一个一个添在空列表里面
s.sort()    //排一下序
print("数据列表",s)  //这个在排序后才打印
print("数据列表最小值",s[0])
print("数据列表最大值",s[-1])
print("数据列表平均值",sum(s)/10)  直接就求平均值了
```
##### 题目二
```
s1=[19,23,54,64,875,20,109,232,123,54]
s2=[55,80,72,35,60,123,54,29,91]
for num in s2:
    s1.append(num)
print(s1)
s=[]
for num in s1:
    if num not in s:
        s.append(num)
print(s)
```
### 解包操作与组包(快速合并列表）
把整体的东西解成一个一个的元素，再放包里处理
s=[*s1,*s2]    //s1,s2中的东西解包后合并

### 列表推导式
#### 通俗解法：
```
s=[]
for num in range(1,21):
    s.append(num*num)
print(s)
```
#### 不带条件的：
```
s2=[i**2 for i in range(1,21)]
print(s2)
```
#### 带条件的：
升级版（需要遍历的）
```
s=[34,2,527,36,58]
s1=[i**2 for i in s if i%2==0]
print(s1)
```


## Dict
### 定义
指键值对（key:value)类型的数据，可以根据key找到对应的值
字典命名={}/
字典名字=dict()
eg:dict1={"王琳":670,"韩立":565}
### 特点：
以键值对储存，键不能重复，可以修改
#### 注意：
字典中的value可以是任何类型的数据，但是key不能为可变类型（如不能是list,set,dict),
key不允许重复，若是重复，后面的覆盖前面的
字典是没有索引下标的，不能根据索引获取值，只可以根据key获取value
获取：v=dict1["王琳"]

### 基本操作
#### 1.添加/修改数据：
字典名称[key]=value
#### 2.删除数据：
eg:score=dict1.pop(key)
eg:del dict1[key]
#### 3.查询数据
eg:dixt1.get(key)
#### 4.获取数据
eg:获取key:dict1.key()
   获取value:dict1.value()
   获取值键对：dict1.items()

### 字典的遍历：
for k in dict1.keys():
     print(f"{k}:{dict1{k}}")
取值键对也可以直接用dict1.item{k}，values()进行遍历

#4.查询购物车：将购物车中的商品信息展示出来，格式为：“商品名称：xxx，商品价格：xxx，商品数量：xxx”。
